Stock price forecast

Multivariate stock price forecast model Daily data is used via Yahoo finance. Model is trained on 80% of the data. Significant lagged relationship between independent and dependent variables is identified. Number of forecast days is equal to number of significant lags. Model is mostly useful for swing traders usually with BTST (Buy today sell tomorrow) approach. Variables important to Value or Long term investors are not included. Link to Web App
May 30, 2021

Signal processing and time series classification (Motion-Classification with 3 classes)

Problem: Fatigue level is a psychological state which is difficult to gauge using non-invasive techniques. Data was taken from an existing research paper where 8 participants performed 3 different activities for 3 hours each. IMU and Heart Rate data was captured independent variables. RPE (Rating of Physical Exertion using Borg’s RPE scale) was used as dependent variable which was collected from each participant at 10 minutes time step. Source data is from the research paper “A data-driven approach to modeling physical fatigue in the workplace using wearable sensors”.
